## Launch files

This folder contains ROS 2 launch files for different teleoperation workflows.

### Available launches

- **URDF display (RViz)**: Visualize the Aero hands in RViz and manipulate joints via a GUI.  
  See `src/launch_files/display_launch/readme.md`.

- **Webcam teleop**: Control the Aero hand using hand pose estimated from a local webcam.  
  See `src/launch_files/webcam_teleop_launch/readme.md`.

- **Apple Vision Pro teleop**: Control the Aero hand using hand tracking streamed from an Apple Vision Pro.  
  See `src/launch_files/vision_pro_teleop_launch/readme.md`.

- **Manus glove teleop**: Control the Aero hand using hand pose captured by a Manus glove.  
  See `src/launch_files/manus_teleop_launch/readme.md`.
