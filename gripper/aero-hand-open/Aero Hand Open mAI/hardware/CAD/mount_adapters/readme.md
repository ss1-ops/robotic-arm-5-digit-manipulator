This folder contains all the adapters for Aero Hand Open designed for different robotic systems.

For instructions on connecting the adapter(s) with Aero Hand Open, please refer to [Adapter_Assembly_Instructions.pdf](Adapter_Assembly_Instructions.pdf).

## Available Adapters

**piper_arm_adapter.stp** - Designed for the [Piper Arm robotic system](https://global.agilex.ai/products/piper).

**threaded_mount_adapter.stl** - Designed to connect with a tripod mount, as demonstrated in our [previous X post](https://x.com/TetherIA_ai/status/1987711095533113664?s=20).

**135degrees_adapter.stl** - A special adapter for the Piper robot arm that provides an optimal grasping angle. See [reference image](assets/135degrees_adapter.png).

**G1_Open_Unitree_Converter.stl** - adapter for [unitree G1 robot](https://www.unitree.com/g1)
