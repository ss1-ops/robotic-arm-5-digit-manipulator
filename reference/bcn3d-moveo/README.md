# BCN3D Moveo — Original Open Source Reference

This directory contains the original mechanical design files (STLs, images, documentation) from the excellent open-source BCN3D Moveo 3D-printed robotic arm project.

## Attribution
- Project: BCN3D Moveo
- Original repository / source: https://github.com/BCN3D/BCN3D-Moveo (or the Thingiverse/Printables equivalent for the specific release 1693444)
- License: See the included LICENSE.txt in each part folder (typically Creative Commons or project-specific open hardware license).

These files are included here **for reference and attribution only**. The active custom work (enclosure, electronics integration, firmware, ROS 2 control, Aero Hand gripper, vision) lives in the sibling directories at the project root.

## Contents
- `part-1/`: Original files (includes images, docs)
- `part-2/`: Original STLs and some modified files (e.g. geared wrist variants)

Do not modify these unless syncing with upstream. Custom modifications are tracked in `hardware/cad/` and the gripper sub-project.
